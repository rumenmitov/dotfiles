# Arch Script GUIDE
*****************************
## Prerequisites
✅ You must be running an Arch system with access to internet  
✅ Replace *Icon=/var/lib/AccountsService/icons/**username**.png* with your username in *lightdm/user.txt*  
## How To Run The Installation
Make sure that you are in the `$HOME` directory and that **dotfiles** is there as well when you run the following command:  
```
cd  
sudo chmod +x dotfiles/scripts/arch.sh  
sudo ./dotfiles/scripts/arch.sh  
```
